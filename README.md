# practice-repo

Custom fields used on the Account Object
- Start Date: start_date__c [DATE]
- End Date: end_date__c [DATE]
- Number of Days: number_of_days__c [NUMBER]

Business Hours
- Name: WeekDays
- Time Zone	(GMT-08:00) Pacific Standard Time (America/Los_Angeles)
- Business hours: 
     | Days | Hours |
     | --- | --- |
     | Sunday | No Hours |
     | Monday | 24 Hours |
     | Tuesday | 24 Hours |
     | Wednesday | 24 Hours |
     | Thursday |	24 Hours |
     | Friday |	24 Hours |
     | Saturday |	No Hours |
